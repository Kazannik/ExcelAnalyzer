Предварительная обработка собранных данных

Приложение Stat for Microsoft Office Excel 2007 является надстройкой программы Microsoft Office Excel, предназначенной для работы с электронными таблицами.

Надстройки используются для автоматизации приложений Microsoft Office, а также расширения функций и настройки пользовательского интерфейса приложения.

Надстройка Stat for Microsoft Office Excel 2007 позволяет автоматизировать сбор и обработку статистических сведений, представленных в таблицах файлов в формате Microsoft Office Excel (*.xls; *.xlsx).

Основные функции надстройки:

Открытие файлов трассировки

Объясняется, как открыть любой поддерживаемый файл трассировки, добавлять или интегрировать дополнительные файлы трассировки, а также открывать и проводить одновременное слияние группы файлов трассировки.

Просмотр данных трассировки

Рассматривается четыре различных представления: Действие, Проект, Сообщение и График.

Фильтрация трассировок

Объясняется, каким образом был упрощен анализ файлов трассировки.

Поиск данных трассировки

Описывается, как можно найти требуемые данные трассировки.

Навигация по данным трассировок

Рассматривается несколько вариантов навигации в трассировках.
