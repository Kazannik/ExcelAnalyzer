Создание коллекции директорий (папок)
Внимание!
Работа функций создание коллекции директорий (папок) зависит от получения доступа к настройкам СПО АРМ "Статистика Генеральной прокуратуры Российской Федерации".

Для создания набора папок с именами соответсвующими наименованиям субъектов Российской Федерации нажмите кнопку Создать набор папок для сбора файлов Работа с файлами в группе элементов управления Обработка файлов на ленте надстройки Stat for Microsoft Office Excel 2007.

Надстройка Stat for Microsoft Office Excel 2007 позволяет автоматизировать сбор и обработку статистических сведений, представленных в таблицах файлов в формате Microsoft Office Excel (*.xls; *.xlsx).

Для автоматического сбора статистических данных необходимо: создать папки в соответсвии с именами субъектов Российской Федерации; в созданные папки поместить файлы в формате Microsoft Office Excel (*.xls; *.xlsx), сформированные прокуратурами субъектов Российской Федерации; сформировать единую книгу Microsoft Office Excel путем объединения файлов, размещенных в папках; при необходимости добавить в книгу Microsoft Office Excel листы для формирования промежуточных результатов; провести суммирование показателей со всех листов сформированной книги Microsoft Office Excel.

Для проведения анализа показателей книги Microsoft Office Excel предусмотрена возможность применения дополнительных инструментов, входящих в состав Stat for Microsoft Office Excel 2007, либо экспортирования полученных данных в формате СПО "Автоматизированное рабочее место "Статистика Генеральной прокуратуры Российской Федерации".

Примечание
Надстройки используются для автоматизации приложений Microsoft Office, а также расширения функций и настройки пользовательского интерфейса приложения.

Основные функции надстройки:

Создание коллекции директорий (папок)
Объясняется, каким образом создать коллекцию директорий (папок), чьи наименования соответствуют названиям подразделений органов прокуратуры.

Осуществление контроля наличия в папках файлов в формате Miсrosoft Office Excel
Объясняется, каким образом проверить наличие в соответствующих папках подразделений органов прокуратуры файлов в формате Microsoft Office Excel (*.xls; *.xlsx).

Импорт сведений из файлов Microsoft Office Excel
Объясняется, каким образом импортировать в единую книгу Microsoft Office Excel сведения из файлов в формате Microsoft Office Excel (*.xls; *.xlsx), находящихся в дирректориях (папках) соответствующих подразделениям органов прокуратуры.

Предварительная обработка собранных данных
Объясняется, каким образом проводится предварительная обработка собранных данных, что включает в себя добавление в единую книгу Microsoft Office Excel отдельных листов, соответствующих промежуточным итогам (Например, «Всего по прокуратурам субъектов РФ»), сортировку листов.

Суммирование статистических сведений
Объясняется, каким образом проводится операция суммирования статистических сведений, расположенных на различных листах книги Microsoft Office Excel.

Экспорт статистических сведений
Объясняется, каким образом проводится экспорт статистических сведений в формате СПО «Автоматизированное рабочее место «Статистика Генеральной прокуратуры Российской Федерации» для дальнейшей обработки и анализа в указанном приложении.

Инструменты, позволяющие проводить оперативный анализ данных
Описывается, как можно провести оперативный анализ статистических данных при помощи специального набора инструментов.

Настройка работы приложения

Внимание!
Работа таких функций приложения как создание коллекции директорий (папок); предварительная обработка собранных данных; суммирование статистических сведений; экспорт статистических сведений зависит от получения доступа к настройкам СПО АРМ "Статистика Генеральной прокуратуры Российской Федерации".

Для настройки доступа приложения Stat for Microsoft Office Excel 2007 к СПО АРМ "Статистика Генаральной прокуратуры Российской Федерации" необходимо:

  1. Нажмите кнопку вызова диалогового окна в группе элементов управления Обработка файлов на ленте надстройки Stat for Microsoft Office Excel 2007.

  2. В открывшемся диалоговом окне Настройки приложения нажмите кнопку Обзор.

  3. В открывшемся диалоговом окне откройте файл STAT_WIN.INI, являющйся файлом настроек СПО АРМ "Статистика Генеральной прокуратуры Российской Федерации".

  4. В диалоговом окне Настройки приложения для подтверждения настроек нажмите кнопку OK.

    Примечание
    Для отказа от применения настроек нажмите кнопку Отмена.
