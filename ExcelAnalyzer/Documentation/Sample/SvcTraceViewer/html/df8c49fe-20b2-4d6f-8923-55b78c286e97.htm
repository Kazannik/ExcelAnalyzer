Поиск данных трассировки

В этом разделе описывается как можно найти необходимые данные трассировки.

Поиск данных трассировки

В средстве просмотра предусмотрены два способа поиска данных:

  • Панель поиска предоставляет быстрый доступ к наиболее распространенным поисковым параметрам.

  • Диалоговое окно поиска содержит больше параметров поиска. Открыть это окно можно в меню Правка или при помощи сочетания клавиш «CTRL + F».

Панель поиска отображается поверх средства просмотра. Если она отсутствует, включите ее в меню Вид. Панель инструментов содержит два компонента:

  • Искать: позволяет ввести поисковый запрос.

  • Диапазон поиска: позволяет указать область поиска. Можно выбрать поиск во всех действиях или только в текущем действии.

Диалоговое окно поиска содержит два дополнительных параметра:

  • Найти объект: параметр «Необработанные данные журнала» позволяет искать ключевое слово среди всех необработанных данных. Параметры «текст XML» и «атрибут XML» позволяют осуществлять поиск только среди элементов XML. Параметр «Сообщение из журнала» позволяет производить поиск ключевого слова только в сообщениях.

  • Пропустить корневое действие: процедура поиска пропускает трассировки в действии «000000000000». Этот параметр увеличивает производительность при работе с большими файлами трассировки, если корневое действие содержит тысячи трассировок, большая часть которых является перенаправлениями.
