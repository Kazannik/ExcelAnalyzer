Программа Service Trace Viewer

Программа Windows Communication Foundation (WCF) Service Trace Viewer помогает анализировать диагностические трассировки действий (SVCLOG), создаваемые прослушивателями WCF. Программа Service Trace Viewer предоставляет возможность простого слияния, просмотра и фильтрации сообщений трассировки, чтобы можно было диагностировать, исправлять и проверять неисправности служб WCF.

Дополнительные сведения о функциональных возможностях трассировки, предоставляемых WCF, см. в разделе Источники информации об инструменте Service Trace Viewer.
