Описание настроек
[Печать границ поля]
Выберите эту настройку при печати линий границы.
[Задать поля для вырезания/вставки]
Выберите эту настройку при печати вырезанного фрагмента или фрагмента с полями и укажите [Ширина границы поля].
Для печати с метками, указывающими на вырезанные фрагменты/фрагменты с полями, выберите [Метки печати для выравнивания].
[Страниц на печать]
Укажите страницу(ы) для печати при печати плаката.
[Все страницы]
Выполняется печать всех страниц плаката.
[Задать страницы]
Выполняется только печать указанных страниц плаката.
