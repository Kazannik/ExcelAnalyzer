Описание настроек
[Профиль]
При выборе профиля применяются зарегистрированные настройки. Это удобно для сохранения часто используемых настроек.
Печать с профилями
[Добавление]
Позволяет зарегистрировать текущие настройки в качестве профиля. Можно установить название и прочие параметры в Диалоговое окно [Добавление профиля].
Добавление профилей
[Правка]
Позволяет корректировать, импортировать и экспортировать профили в Диалоговое окно [Правка профиля].
Правка списка профилей
[Способ вывода]
Позволяет выбрать метод вывода данных для печати.
[Печать]
Выполняет обычную операцию печати.
[Удержание]
Временно сохраняет данные для печати в очереди удержания принтера. Эта функция удобна, если есть необходимость выполнить пробную распечатку. Наименование данных можно установить в Диалоговое окно [Детали удержания], нажав [Подробно].
Приостановление процесса печати
[Защищенная печать]
Закрепляет за данными для печати a PIN-код (пароль) и сохраняет его в принтере. Эта функция удобна при печати конфиденциальных документов и др. Имя пользователя и PIN-код или пароль можно установить, нажав [Подробно].
Защищенная печать
[Сохранение]
Позволяет сохранять данные для печати в окне принтера. Место сохранения можно установить, нажав [Подробно] (Диалоговое окно [Сохранение деталей]).
Сохранение данных для печати
[Активная печать]
Выполняет задание на печать вне очереди, после того, как будет закончено выполнение текущего задания на печать.
[Правка и предварительный просмотр]
Объединяет несколько файлов в один для распечатки. Можно также просмотреть данные перед печатью и изменить настройки печати. Если выбрано [Блокировать], [Способ вывода] блокируется в режиме [Правка и предварительный просмотр]. Эта функция полезна, если нужно ограничить методы вывода для клиентов в среде общего принтера.
Объединение нескольких файлов
[Блокировать] можно установить только в том случае, если экран драйвера принтера открыт из меню Пуск (см. Изменение настроек по умолчанию).
В начало
[Выбор по]
Позволяет выбрать источник бумаги и тип бумаги для печати.
Указание источников бумаги
Указание типов бумаги
[Выбор бумаги]
Можно выбрать метод подачи (источник) бумаги в зависимости от страницы. Если выбрано [Типу бумаги] для [Выбор по], типы бумаги можно выбирать нажатием [Настройки].
[Одинаковая бумага для всех страниц]
Позволяет установить источник бумаги или тип бумаги для всех страниц.
[Разные для первой, других и последней страниц], [Разные для первой, второй, других и последней страниц]
Позволяет установить источники бумаги или типы бумаги для каждой указанной страницы.
[Разные для обложки и других страниц]
Позволяет установить источники бумаги или типы бумаги для обложки и других страниц.
Необходимое условие: выберите вкладку [Основные настройки] → [Печать односторонняя/двухсторонняя/буклет] → [Печать буклета],
или вкладку [Окончательная обработка] → [Стиль печати] → [Печать буклета]
[Вставка листов]
Вставка листов бумаги между указанными страницами. Нажмите [Настройки вставки листа] чтобы задать настройки, относящиеся к вставленным листам. Выберите исходную бумагу в [Исходный источник бумаги] или [Исходный Типу бумаги].
Диалоговое окно [Настройки вставки листа]
[Прозрачные титульные листы]/[Прокладывание кальки между листами]
При выполнении печати на прозрачной пленке, между листами прозрачной пленки вставляются листы бумаги. Выберите источник или тип бумаги для [Титульные листы]/[Прокладочные листы] и [Кальки]. При выборе [Печать на титульных листах]/[Печать на прокладочных листах] печать выполняется и на прокладочных листах.
Необходимое условие: выберите вкладку [Основные настройки] или [Макет страницы] → [Размер вывода] → [A4] или [Letter]
Печать на обеих сторонах пленки невозможна. Если установлен этот параметр, выберите [Односторонняя печать] на вкладке [Основные настройки] или [Окончательная обработка]. Сведения о переключении между 2-сторонней и 1-сторонней печатью см. в разделе Односторонняя и двухсторонняя печать.
В начало
[Вертикальная подача специальной бумаги]
Бумага специальных форматов при печати подается вертикально.
[Вертикальная подача бумаги A4/Letter]
Бумага форматов A4 и Letter при печати подается вертикально.
[Горизонтальная подача специальной бумаги]
При печати бумага специальных форматов подается в горизонтальной ориентации.
[Титульные листы]/[Прокладочные листы]
Позволяет выбрать источник бумаги или тип бумаги прокладочных листов, используемых для стабилизации прозрачной пленки. Щелчок на [Настройки] также позволит выбирать различные типы бумаги, зарегистрированные в принтере.
Данный параметр доступен только том случае, если [Прозрачные титульные листы]/[Прокладывание кальки между листами] было выбрано для [Выбор бумаги].
[Печать на титульных листах]/[Печать на прокладочных листах]
Печатает одно и то же изображение на прокладочных листах и кальке, если в разделе [Прозрачные титульные листы]/[Прокладывание кальки между листами] выбрано значение [Выбор бумаги].
[Печатать на другой стороне]
Если нужно печатать на обеих сторонах бумаги, подаваемой вручную в уравнительный лоток или универсальный лоток, выберите эту функцию для печати на обратной стороне листа.
[Сведения о бумаге для таблиц]
Позволяет установить позицию печати на бумаге для таблиц в Диалоговое окно [Сведения о бумаге для таблиц].
Необходимое условие: Выберите вкладку [Источник бумаги] → [Типу бумаги] → бумага для таблиц
[Настройки передней/задней стороны обложки]
Вставка передней и задней обложек. Можно установить источник бумаги или тип бумаги для передней и задней обложек, а так же установить, следует ли на них печатать, в Диалоговое окно [Настройки вставки листа].
[Восс. параметры по умолчанию]
Восстанавливает стандартные (по умолчанию) значения всех настроек.
[Настройки вида]
Позволяет проверить текущие настройки в списке.
[Отображение окна состояния принтера]
Для отображения окна состояния принтера нажмите кнопку []. Окно состояния принтера позволяет проверять состояние принтера, выполнять служебные функции, задавать параметры модуля принтера и т.п.
Подробнее об окне состояния принтера см. в справке, отображаемой с помощью строки меню [Справка].
