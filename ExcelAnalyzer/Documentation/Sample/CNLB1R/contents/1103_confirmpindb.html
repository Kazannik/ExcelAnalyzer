Диалоговое окно [Подтвердить PIN-код]/[Подтвердить пароль]
Позволяет установить PIN-код/пароль для выполнения защищенной печати. Для запуска процесса печати выберите наименование данных для печати при помощи панели управления и введите PIN-код/пароль.
Описание в Функции, предлагаемые на всех вкладках (настройки)
Процедура для Защищенная печать
Описание настроек
[Имя документа]
Позволяет установить наименование данных для печати.
[Имя пользователя]
Позволяет установить имя пользователя для данных для печати. Имя пользователя устанавливается в Диалоговое окно [Настройки информации пользователя], но его можно изменить.
Невозможно изменить имя пользователя, если на вкладке [Настройки устройства] выбран пункт [Проверка подлинности пользователя] - [Управление пользователями].
[PIN-код]/[Пароль]
Позволяет установить PIN-код или пароль. Если PIN-код/пароль задан в Диалоговое окно [Настройки информации пользователя], он отображается как ряд точек, но его можно изменить.
