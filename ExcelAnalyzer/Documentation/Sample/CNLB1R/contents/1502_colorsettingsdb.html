Описание в Вкладка [Качество] (Настройки)
Описание настроек
Вкладка [Регулировка цвета]/[Регулировка оттенков серого]
Можно сравнить отрегулированное изображение с [Оригинал] и [Предварительный просмотр].
[Добавить (Цвет)]
Нажмите на кнопку цвета, который Вы желаете усилить. Чтобы отрегулировать цвет, щелкните на изображении и перетащите его в направлении цвета(ов), которые Вы желаете усилить.
[Яркость]
Настройка яркости при помощи ползунка.
[Контраст]
Настройка контраста при помощи ползунка.
[Восс. параметры по умолчанию]
Восстанавливает стандартные (по умолчанию) значения всех настроек.
Вкладка [Сопоставление]
Позволяет задать настройки, связанные с сопоставлением и настройкой коэффициента контрастности.
[Режим сопоставления]
[Настройки принтера по умолчанию]
Используется настройка принтера по умолчанию.
[Режим сопоставления драйвера]
Выполняется сопоставление драйвера принтера (компьютера).
[Режим сопоставления устройства]
Выполняется сопоставление на принтере.
[Режим ICM]
Выполняется сопоставление с использованием функции ICM драйвера принтера.
[Регулировка коэффициента контрастности]
Яркость результата печати можно отрегулировать, чтобы сохранить самые светлые и самые темные участки изображения. Эта функция полезна, если изображение нужно распечатать ярче оригинала или поменять яркость при печати. Выберите значение коэффициента контрастности в [Коэффициент контрастности].
Если используется высокопроизводительный компьютер, рекомендуется провести настройку на хост-компьютере. Это может помочь сократить время печати.
[Режим сопоставления света окружающей среды [Kyuanos]]
Выполняется сопоставление изображений, отрегулированных в соответствии с различиями освещения. Настройте [Цветовое пространство монитора] и [Внешний свет] в [Режим сопоставления света окружающей среды [Kyuanos]].
[Без сопоставления драйвера]
Выберите эту функцию для выполнения сопоставления в приложении. Если сопоставление выполняется как в приложении, так и в принтере или драйвере, цвета могут печататься неправильно.
[Использование]
Выберите эту функцию для выполнения сопоставления в драйвере (компьютере) или на принтере.
[Не использовать]
Выберите эту функцию для выполнения сопоставления в приложении.
[Приоритет сопоставления цветов приложения]
Если печатаемые цвета не соответствуют желаемым, сняв данный флажок можно выполнить сопоставление цветов на драйвере или на устройстве.
[Метод сопоставления]
Указывает, какой элемент должен быть приоритетным при сопоставлении цветов.
[Фотографический]
При сопоставлении предоставляет приоритетность цветовым оттенкам.
[Перцепционный [Фотографический]]/[Общие положения]
При сопоставлении предоставляет приоритетность цветовым оттенкам и генерирует фотографический тон.
[Перцепционный [Подстройка цветов монитора]]/[Перцепционный]
При сопоставлении обеспечивается приоритет цветовых тонов и подстройка цветов монитора.
[Презентация]/[Насыщенность]
Предоставляет приоритет яркости в соответствии.
[Относительный цветометрический]/[Цветометрический]
Приоритет отдается сокращению разницы между цветами оригинала на экране (например, логотипа) и отпечатка.
[Абсолютный цветометрический]/[Цветометрический [Без коррекции точки белого]]
Сопоставление функций, аналогичных Относительному цветометрическому параметру и Цветометрическому параметру, но без коррекции точки белого.
[Vivid Photo]
Обеспечение более насыщенных и глубоких цветов, чем при настройке [Перцепционный [Фотографический]].
[Стандартный]
Использует стандартное сопоставление цветов, подходящее для выбранного объекта.
Функции, недоступные в сочетании с данной функцией:
[Режим сопоставления] → [Настройки принтера по умолчанию]
[Режим сопоставления] → [Не использовать]
[Профиль монитора]
Позволяет выбрать оптимальную настройку для монитора, если в [Режим сопоставления драйвера] выбрано значение [Режим сопоставления]. Если для одного параметра предлагается несколько значений коэффициента контрастности, для получения наилучших результатов выберите более высокое значение коэффициента контрастности, если напечатанное изображение светлее, чем изображение на мониторе.
Если монитор настроен пользователем, при выполнении соответствия очень сложно распечатать цвета, чтобы они соответствовали цветам документа на мониторе.
Функции, недоступные в сочетании с данной функцией: [Режим сопоставления] → [Не использовать]
[Режим сопоставления света окружающей среды [Kyuanos]]
Позволяет выбрать соответствующую настройку, если [Режим сопоставления света окружающей среды [Kyuanos]] было выбрано из [Режим сопоставления].
[Цветовое пространство монитора]
Выберите цветовое пространство вашего монитора.
[Внешний свет]
Выберите цветовую температуру внешнего света.
[Коэффициент контрастности]
Позволяет выбрать соответствующее значение коэффициента контрастности, если [Регулировка коэффициента контрастности] было выбрано из [Режим сопоставления].
[Восс. параметры по умолчанию]
Восстанавливает стандартные (по умолчанию) значения всех настроек.
