Диалоговое окно [Детали удержания]
Позволяет установить наименование данных для печати, если в качестве метода вывода выбрано удержание. Данные для печати удерживаются в очереди удерживания (очередь ожидающих распечатки данных) в принтере. Данные выводятся согласно инструкциям, подаваемым с панели управления принтера или с помощью удаленного пользовательского интерфейса.
