Указание источников бумаги
Страницу для печати можно указать по источнику бумаги.
1.
Откройте вкладку [Источник бумаги].
Если используется принтер серии LBP, переходите к шагу 3.
2.
Выберите [Выбор по] → [Источнику бумаги].
3.
Если для разных страниц нужно указать разные источники бумаги, выберите желаемую установку в [Выбор бумаги].
4.
Установите источник бумаги для каждой страницы.
