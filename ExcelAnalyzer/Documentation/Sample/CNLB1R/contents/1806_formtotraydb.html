Диалоговое окно [Формат для назначения лотка]
Позволяет подтвердить и выбрать формат и тип бумаги, установленные в каждом источнике бумаги.
Описание в Вкладка [Настройки устройства] (Настройки)
Процедура для Настройки источника бумаги
Описание настроек
[Сведения об источнике бумаги]
Позволяет проверить источники бумаги принтера и узнать, какие форматы бумаги в них заданы. Если информацию о принтере Вы получили при помощи [Опред-е сост. устройства] во вкладке [Настройки устройства], информация о принтере будет отображена здесь. Если информацию о принтере получить невозможно, выберите из списка источник бумаги, затем выберите формат бумаги в [Формат бумаги]. Чтобы установить формат бумаги типа R (например A4R), выберите [Подача с короткого края].
[Типу бумаги]
Позволяет выбрать все типы бумаги, заданные в источниках бумаги. Выбранные типы бумаги отображаются как источники бумаги во вкладке [Источник бумаги], если [Типу бумаги] выбран для [Выбор по].
[Выбор типа загруж. бум. только после получения статуса уст-ва]
Если выбрать эту настройку, при получении информации в пункте [Типу бумаги] можно выбрать бумагу только тех типов, которые загружены в принтер.
