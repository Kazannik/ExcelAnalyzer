Диалоговое окно [Сведения о бумаге для таблиц]
Позволяет установить способ печати на бумаге для таблиц.
Описание в Вкладка [Источник бумаги] (Настройки)
Процедура для Настройки бумаги для таблиц
Описание настроек
[Смещение позиции печати]/[Сдвинуть позицию печати]
Позволяет настроить положение данных для печати в области таблиц бумаги для таблиц. Укажите расстояние смещения данных в [Ширина смещения]/[Ширина сдвига].
Данные для печати создаются для формата бумаги, который не включает в себя область для таблиц.
[Не смещать позицию печати]/[Не сдвигать позицию печати]
Печать данных, созданных для специального формата бумаги, включающего в себя область для таблиц. Выберите A4 или Letter в [Формат бумаги для таблиц].
Данные для печати создаются для специального формата бумаги, включающего в себя область для таблиц (бумага для таблиц A4 = 223 x 297 мм (8,77 x 11,69 дюйма), бумага для таблиц Letter = 228,6 x 279,4 мм (9,00 x 11,00 дюймов)).
