Диалоговое окно [Настройки проверки подлинности пользователя]
Позволяет задать пароль и пр. для использования при проверке подлинности пользователя.
Описание в Вкладка [Настройки устройства] (Настройки)
Процедура для Проверка подлинности пользователя
Описание настроек
[Разрешить настройки пароля]
Выберите этот пункт, задавая пароль. Если не задать пароль здесь, введите пароль при печати.
[Имя пользователя]
Отображается имя пользователя, заданное в диалоговом окне [Настройки информации пользователя].
[Пароль]
Если выбран пункт [Разрешить настройки пароля], задайте пароль.
[Проверка]
Проверяются настройки принтера.
[Подтвердить сведения для проверки подлинности во время печати]
При печати отображается диалоговое окно для подтверждения информации по проверке подлинности.
[Выполнить проверку подлинности на устройстве]
Выполняется проверка подлинности на устройстве.
