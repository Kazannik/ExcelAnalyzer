Описание в Вкладка [Качество] (Настройки)
Описание настроек
[Графический режим]
Позволяет выбрать метод обработки данных для печати.
[Режим UFR II]
Обработка отправляемых на принтер данных для печати с помощью команд управления UFR II. Выбирайте данный режим в обычных случаях.
[Растровый режим]
Драйвер рисует линии, круги и т.д. в драйвере принтера, после чего отправляет готовое изображение на принтер.
Это может быть медленнее, чем режим UFR II, и может привести к отправке на принтер большего количества данных.
[Скорость обсчета изображений]
Позволяет задать способ растрирования изображения для преобразования цветных документов в документы в полутонах серого.
[1 BPP]
Заменяет на черные все, кроме белых, части символов и тонких линий, и печатает на высокой скорости.
[1 BPP [Улучшенный]]
Выражает градации серого полутонами и печатает на высокой скорости.
[24 BPP]
Печать в этом режиме занимает больше времени, чем в режимах [1 BPP] и [1 BPP [Улучшенный]], при этом качество распечатываемых документов очень высоко.
[Разрешение]
Позволяет выбрать разрешение печати.
[Градация оттенков]
Установка градации для печати растровых изображений. [Высокая 2] использует более тонкие градации, чем [Высокая 1], и распечатывает очень плавные градации.
[Растровые изображения] (Цвет)
Позволяет настроить растрирование шаблонов для соответствующих полутонов (областей между темными и светлыми участками).
[+][-]
[Настройки принтера по умолчанию]
Используются настройки принтера.
[Разрешение]
Применение растрирования шаблонов с приоритетом разрешения. Равномерная печать контуров. Данная настройка подходит для печати стандартных документов и графических изображений (текста, графиков и др.).
[Градация оттенков]
Применение растрирования шаблонов с приоритетом градации. Данная настройка подходит для печати со сглаженной градацией.
[Диффузия ошибок]
Применение растрированного шаблона с приоритетом диффузии ошибок, что прекрасно подходит для детализации. Данная настройка подходит для печати мелкого текста и тонких линий.

Необходимое условие: выберите вкладку [Качество] → [Дополнительные настройки] → [Графический режим] → [Режим UFR II] (также [Растровый режим] где была выбрана вкладка [Качество] → [Дополнительные настройки] → [Скорость обсчета изображений] → [1 BPP [Улучшенный]] или [24 BPP])
[Калибровка с хост-компьютера]
Определяет, выполняется ли калибровка графических данных хост-компьютером или принтером. Это позволяет печатать более четкие изображения, однако снижает скорость обработки данных.
[Авто]
Калибровка с хост-компьютера устанавливается автоматически.
[Вкл.]
Делает возможной настройку калибровки с хост-компьютера.
[Выкл.]
Делает настройку калибровки с хост-компьютера невозможной.
Функции, недоступные в сочетании с данной функцией: [Качество] → [Дополнительные настройки] → [Графический режим] → [Растровый режим]
[Получить калибровочную информацию при печати]
Получает калибровочную информацию от принтера каждый раз при печати документа.
[Экономия тонера]
Уменьшает плотность печати данных для экономии тонера.
Функции, недоступные в сочетании с данной функцией: [Плотность тонера] → [Регулировать]
[Плотность тонера]
Позволяет задавать плотность тонера. Данная функция полезна, если нужно экономить тонер и регулировать плотность тонера при печати баркодов. Если отображается [Подробно], можно отрегулировать плотность и выполнить точную настройку плотности тонера каждого цвета в диалоговом окне [Сведения о плотности тонера].
При выборе [Настройки принтера по умолчанию] можно получить нежелательные результаты печати, в зависимости от настроек принтера. Например, выбрав [Настройки принтера по умолчанию], результат на выходе может быть более светлым, чем с настройкой [-1 [Светлее]].
В зависимости от используемой модели принтера, если для параметра [Способ вывода] выбрано значение [Сохранение] или [Удержание], невозможно выполнить точную настройку плотности тонера в пункте [Подробно].
Функции, недоступные в сочетании с данной функцией: [Экономия тонера] → [Вкл.]
[Регулировка плотности тонера]
Позволяет регулировать плотность тонера каждого цвета. Выбрав [Вкл.] и нажав [Подробно], можно отрегулировать плотности тонеров цветов [Голубой], [Малиновый], [Желтый] и [Черный], перемещая движки.
Функции, недоступные в сочетании с данной функцией: Вкладка [Качество] → [Дополнительные настройки] → [Экономия тонера]
[Точная регулировка плотности тонера]
Позволяет регулировать плотность тонера по тоновому спектру. Выбрав [Вкл.] и нажав [Подробно], можно отрегулировать плотности тонеров по тоновым спектрам [Высокая], [Средняя] и [Низкая], перемещая движки. Можно отрегулировать плотность тонеров, вводя числа.
[Дополнительное сглаживание]
Сглаживает контуры текста и линий, распечатывает плавные градации. [Плавные переходы 2] выполняет сглаживание более интенсивно, чем [Плавные переходы 1].
[Настройка уровня тонера]/[Коррекция уровня тонера]
Позволяет выбрать способ настройки уровня тонера.
[Настройки принтера по умолчанию]
Используются настройки принтера.
[Нормальный]
Используется стандартное количество тонера.
[Приоритет градации]
Используется оптимальное количество тонера для документов, содержащих много графики или компьютерной графики, содержащей градацию.
[Приоритет текста]
Используется оптимальное количество тонера для документов, содержащих преимущественно текст.
[Управление линиями]
Улучшает воспроизводимость текста и линий. Выберите [Приоритет разрешения], чтобы отдать приоритет разрешению, и [Приоритет градации], чтобы отдать приоритет градации.
[Резкость]
Позволяет настраивать резкость контуров изображений.
[Резкость (фото)]
Задайте способ обработки для краев изображений с низким разрешением. Процессы, которые можно задать, различаются для различных моделей принтеров.
[Ускоренная обработка изображения]
Позволяет указать степень используемой высокоскоростной обработки. Однако высокая скорость обработки может привести к снижению качества печати.
[Точная регулировка плотности]
Позволяет настраивать точную настройку плотности для текста небольшого размера и тонких линий. Данная функция полезна при исправлении "поломок" в тексте и линиях.
Необходимое условие:
Выберите вкладку [Качество] → диалоговое окно [Дополнительные настройки] → [Режим регулировки вывода] → [Выкл.]
Выберите вкладку [Качество] → диалоговое окно [Дополнительные настройки] → [Разрешение] → [1200 dpi]
[Обработка сжатых изображений]
Возможность выбора способа обработки печати, который будет использоваться при печати сжатых графических данных.
Функции, недоступные в сочетании с данной функцией:
Вкладка [Качество] → [Дополнительные настройки] → [Графический режим] → [Растровый режим]
Вкладка [Качество] → [Дополнительные настройки] → [CanoFine] (если выбрано значение [Обработка сжатых изображений] → [Устройством])
[Авто]
Автоматически выбирает оптимальный способ обработки печати.
[На хост-компьютере]
Восстанавливает сжатые изображения на хост-компьютере.
[Устройством]
Восстанавливает сжатые изображения на устройстве.
Как правило, это позволяет выполнять печать с высокой скоростью.
[Совместная обработка изображений]
Возможность объединения подробных графических данных на хост-компьютере.
[Выравнивание серого]
Возможность надежной печати серых и черных областей (R=G=B) серым или черным цветом.
[Сглаживание градации]
Равномерно печатает переходы между цветами в градациях. [Плавные переходы 2] выполняет сглаживание более интенсивно, чем [Плавные переходы 1].
[Сжатие изображения]
Позволяет отрегулировать коэффициент сжатия отправляемых на принтер данных для печати. Можно отдать приоритет изображению или скорости печати.
[Печать диагональных линий с высокой скоростью]
Сокращает время на обработку при печати диагональных линий. В результате между диагоналями на графиках могут появиться промежутки. Если это происходит, выключите эту функцию.
[Оптимизация сжатия изображения]
Служит для переключения способов сжатия данных при их отправке на принтер.
[Специальный режим 1 BPP]
Позволяет уменьшить размер данных для печати.
Необходимое условие:
Выберите вкладку [Качество] - [Дополнительные настройки] - [Графический режим] - [Режим UFR II]
Выберите вкладку [Качество] - [Дополнительные настройки] - [Скорость обсчета изображений] - [1 BPP]
Выберите вкладку [Окончательная обработка] - [Дополнительные настройки] - [Всегда растрировать на хост-компьютере] - [Выкл.]
[Режим регулировки вывода]
Переключает разрешение печати 600 x 600 dpi и 600 x 400 dpi.
[Регулировка ширины]
Отрегулируйте толщины конкретных объектов, например текста и линий.
[Захват краски]
Выполняет перекрывание границ текстом или графикой и фоновым изображением, чтобы избежать появления белых линий по краям текста или графики. Выбрав [Вкл.], нажмите [Подробно], чтобы задать ширину и плотность захвата краски.
Захват краски не эффективен для светлых символов или графики.
[Настройка уровня тонера для цветной печати]
Настроив количество используемого тонера при цветной печати, можно предотвратить его чрезмерное расходование и возникновение следующих ошибок.
Возникновение пятен тонера на темных участках изображения
Остаточное изображение темного текста или линий
Ошибки фиксации тонера при работе при низких температурах
Затемнение при проецировании изображений, печатаемых на прозрачных пленках
Протекание при копировании двусторонних оригиналов
[Выкл.]
Печать с обычным количеством тонера.
[Уровень 1]
При печати слегка регулирует (уменьшает) количество тонера в областях с насыщенным цветом.
[Уровень 2]
При печати регулирует (уменьшает) расходуемое количество тонера.
[Используйте преобразование оттенков серого для драйвера во время ч/б печати]
Если для параметра [Цветной режим] на листе [Основные настройки]/[Качество] выбрано значение [Черно-белый], печать выполняется с приоритетом настроек черно-белой печати драйвера принтера над настройками приложения.
[CanoFine]
Автоматически регулирует контраст и цветовой баланс изображений для отчетливости печатаемых изображений. Данная функция полезна для корректировки темных фотографий для придания им четкости и яркости. Выбрав [Вкл.], нажмите [Подробно], чтобы задать расширенные настройки.
[Постраничная пакетная обработка]
Обрабатывает несколько точечных рисунков на странице как один элемент данных. Этот флажок следует установить для печати изображений, повернутых под определенным углом, или изображений с вырезами.
[Увеличенная насыщенность]
Усиливает интенсивность цвета при обработке.
[Коррекция положения шаблона]
Коррекция позиции шаблонов.
[Разрешение шаблона]
Задание разрешения для рисования шаблонов.
В начало
