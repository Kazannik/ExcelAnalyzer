Описание настроек
[Процесс печати буклета]
[С настройками приложения]
Выбирайте это значение при определении настроек печати из приложения.
[С настройками драйвера]
Выбирайте это значение, если для используются функции драйвера - разделение документа на наборы, настройки поля переплета и др.
[С настройками устройства]
Выбирайте это значение, если используется функция коррекции сползания (смещения).
[Метод печати буклета]
[Все страницы одновременно]
Все страницы печатаются и сгибаются поперек.
[Разделение на наборы]
Страницы печатаются и сгибаются поперек наборами, согласно указанному количеству страниц.
[Открытие книги]
Выберите этот параметр для указания направления, в котором открывается буклет.
[Указать поле переплета буклета]
Выбирайте это значение, если нужно установить поле переплета и указать его ширину.
В начало
[Использовать коррекцию сползания (смещения)]
Позволяет скорректировать смещение позиции печати между внешними и внутренними страницами, обусловленное толщиной листов бумаги. Кроме того, выравнивается область печати и поля каждой страницы при обрезке краев при помощи торцового станка.
[Способ коррекции смещения]
[Применить настройки устройства]
Используются настройки принтера.
[Автоматическая коррекция]
Смещение для каждой страницы корректируется автоматически.
[Ручная коррекция]
Позволяет указать ширину смещения области печати между внешней и расположенной в середине страницами в качестве значения для коррекции.
[Выравнивание переплета]
Если буклет печатается с использованием функции переплета внакидку, страницы прессуются для разглаживания. Силу прессования можно регулировать, выбрав значение для [Регулировка выравнивания переплета].
Если количество страниц в наборе не превышает 10, операцию прессования можно отменить на принтере.
