Диалоговое окно [Расширенные настройки] (Вкладка [Настройки устройства])
Позволяет задать дополнительные настройки драйвера, такие как специальные сведения о бумаге, названия заданий для данных печати и т.д.
Описание в Вкладка [Настройки устройства] (Настройки)
Описание настроек
[Отображать специальные форматы бумаги, не включенные в этот драйвер]
Отображает список специальных форматов бумаги, размеры полей которых не соответствуют диапазону драйвера.
[Сократить имя задания]
Сокращает имя задания данных печати на панели управления принтера. Если имя задания включает перечисленные ниже префиксы, они будут удалены.
Microsoft Word -
Microsoft PowerPoint -
Microsoft Office Outlook -
Microsoft Outlook -
[Определить атрибуты специального типа бумаги в процессе печати]
Подтверждает соответствие установленного пользователем типа бумаги и типа(ов) бумаги в принтере во время печати.
[Установка для драйвера приоритетных настроек окончательной обработки]
Выполняется печать с приоритетом настроек окончательной обработки, установленных с помощью драйвера.
[Включить специальные настройки CMS]
Позволяет переключать профили, используемые для сопоставления.
[Разрешить только черно-белую печать]
При выборе этой настройки документы обычно печатаются черно-белыми.
[Защищенная печать/Подтверждение управления именем подразделения]
Выберите метод аутентификации для защищенной печати/управления именем подразделения в пунктах [Режим 1] и [Режим 2]. Метод должен совпадать с методом аутентификации, заданным на принтере.
[Использовать прежние данные коррекции градации]
Выберите это, если хотите использовать предыдущие данные коррекции градации.
