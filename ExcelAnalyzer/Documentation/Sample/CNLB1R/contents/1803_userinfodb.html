Описание в Вкладка [Настройки устройства] (Настройки)
Процедура для Настройки информации пользователя
Описание настроек
[Имя пользователя]
Позволяет задать имя пользователя для защищенной печати. Можно ввести до 32 символов.
[PIN-код для защищенной печати]/[Пароль для защищенной печати]
Позволяет задать PIN-код или пароль для защищенной печати. Можно ввести числовой PIN-код от 1 до 9999999.
[Всегда подтверждать сведения о пользователе при выполнении защищенной печати]
Отображает диалоговое окно подтверждения имени документа, имени пользователя и PIN-кода или пароля при выполнении защищенной печати.
[Подтверждать имя пользователя во время печати]
Если выбрана данная установка, при печати отображается диалоговое окно [Подтверждение информации пользователя] а печать выполняется после введения имени пользователя.
[Отображать введенную ранее информацию во время подтверждения]
При выборе данной настройки последнее введенное Вами имя пользователя и/или PIN-код автоматически вводится в диалоговом окне [Подтверждение информации пользователя] или диалоговом окне [Подтвердить PIN-код], которое отображается при печати.
[Управление именем пользователя]/[Настройки значений по умолчанию]
Позволяет задавать настройки по умолчанию для имени пользователя и PIN-кода (пароля).
Необходимое условие: Компьютер с разрешением на полный доступ к принтеру.
[Имя, установленное в качестве имени пользователя]
Позволяет указать имя для входа в Windows, имя компьютера, или любое желаемое имя в качестве имени пользователя.
[PIN-код для защищенной печати]/[Пароль для защищенной печати]
Позволяет ввести PIN-код или пароль длиной до 7 символов.
[Всегда подтверждать сведения о пользователе при выполнении защищенной печати]
Отображает диалоговое окно подтверждения имени документа, имени пользователя и PIN-кода/пароля при выполнении защищенной печати.
[Подтверждать имя пользователя во время печати]
Если выбрана данная установка, при печати отображается диалоговое окно [Подтвердить имя пользователя] а печать выполняется после введения имени пользователя.
[Не разрешать изменять имя пользователя]
Запрещает вносить изменения в имя пользователя в диалоговом окне [Настройки информации пользователя].
[Фиксировать настройку [Настройки информации пользователя]-[Всегда подтверждать сведения о пользователе при выполнении защищенной печати]]
Фиксирует настройку [Всегда подтверждать сведения о пользователе при выполнении защищенной печати] с настройкой, заданной в диалоговом окне [Настройки значений по умолчанию].
[Фиксировать настройку [Настройки информации пользователя]-[Подтверждать имя пользователя во время печати]]
Фиксирует настройку [Подтверждать имя пользователя во время печати] с настройкой, заданной в диалоговом окне [Управление именем пользователя]/[Настройки значений по умолчанию].
