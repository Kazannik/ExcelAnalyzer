Изменение настроек по умолчанию
Изменить настройки по умолчанию можно, открыв экран драйвера принтера из меню Пуск. Такой метод удобен для настройки часто используемых функций (односторонняя или двухсторонняя печать, печать двух страниц на одном листе бумаги и др.).
1.
Нажмите [] (Пуск) → [Устройства и принтеры].
Выше указаны действия в ОС Windows 7 и Server 2008 R2.
В ОС Windows 10: нажмите [Параметры] → [Устройства] → [Принтеры и сканеры] → [Сопутсвующие параметры] → [Устройства и принтеры].
В ОС Windows 8, 8.1, Server 2012 и Server 2012 R2: на рабочем столе выдвиньте боковую панель с правой стороны экрана и выберите пункты [Параметры] → [Панель управления] → [Устройства и принтеры].
В ОС Windows Vista и Server 2008: нажмите [] (Пуск) → [Панель управления] → [Принтеры].
В ОС Windows Server 2003 и Server 2003 R2: нажмите [Пуск] → [Принтеры и факсы] (или [Пуск] → [Панель управления] → [Принтеры и другое оборудование] → [Принтеры и факсы]).
2.
Щелкните правой кнопкой мыши значок используемого принтера → нажмите [Настройки печати].
Отобразится экран параметров печати драйвера принтера.
3.
Настройте необходимые функции → нажмите [ОК].
Переходя с вкладки на вкладку, настройте необходимые функции.
Функции, предлагаемые на всех вкладках (что можно сделать)
Вкладка [Основные настройки] (что можно сделать)
Вкладка [Макет страницы] (что можно сделать)
Вкладка [Окончательная обработка] (что можно сделать)
Вкладка [Источник бумаги] (что можно сделать)
Вкладка [Качество] (что можно сделать)
