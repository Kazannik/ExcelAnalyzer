Уменьшенный макет
Можно печатать по несколько страниц документа на каждой стороне каждого листа бумаги.
Печать данных с нескольких страниц на одной странице
О данной функции
1.
Откройте вкладку [Основные настройки] или [Макет страницы].
2.
Выберите [Разметка страницы] → [N на 1] (N = 2, 4, 6, 8, 9 или 16).
N соответствует количеству страниц, размещаемых на каждой стороне листа бумаги. Например, при выборе настройки [4 на 1] на одной стороне листа бумаги печатаются по 4 страницы документа.
Сведения о настройке [Плакат [1 x 2]] или [Плакат [N x N]] приведены в разделе "Печать плакатов".
3.
Выберите порядок расположения печатаемых страниц в [Порядок распол. страниц].

О данной функции
Функции, недоступные в сочетании с данной функцией
Вкладка [Основные настройки] или [Макет страницы] → [Ручная настройка масштаба]
[N на 1] (N = 6, 8, 9, 16): [Настройки устройства] → [Буферизация на хост-компьютере] → [Отключена]
(кроме [2 на 1] и [4 на 1])
Справки
Если в приложении для распечатки был выбран метод сортировки страниц, документ может быть распечатан неправильно.
Если данная функция используется совместно с функцией печати с наложением, документ может быть распечатан неправильно.
