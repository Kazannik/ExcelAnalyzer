Диалоговое окно [Добавление/изменение профиля]
Позволяет определять настройки нового добавляемого профиля или зарегистрированного пользователем профиля.
Описание в Вкладка [Профиль] (Настройки)
Процедура для Добавление клиентских профилей
Процедура для Правка списка клиентских профилей
Описание настроек
Вкладка [Присвоение атрибутов]
[Имя]
Позволяет ввести название профиля.
[Значок]
Позволяет выбрать значок и закрепить его за профилем.
[Комментарий]
Позволяет ввести соответствующий комментарий. Комментарий отображается при выборе профиля в списке [Профиль]. Комментарий удобен для введения описания настроек.
Вкладка [Макет страницы]/[Окончательная обработка]/[Источник бумаги]/[Качество]
Позволяет устанавливать функции, которые будут зарегистрированы в качестве профиля.
