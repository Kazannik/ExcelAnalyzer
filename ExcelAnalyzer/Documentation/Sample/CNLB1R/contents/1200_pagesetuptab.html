Описание настроек
[Профиль]
При выборе профиля применяются зарегистрированные настройки. Это удобно для сохранения часто используемых настроек.
Печать с профилями
[Добавление]
Позволяет зарегистрировать текущие настройки в качестве профиля. Можно установить название и прочие параметры в Диалоговое окно [Добавление профиля].
Добавление профилей
[Правка]
Позволяет корректировать, импортировать и экспортировать профили в Диалоговое окно [Правка профиля].
Правка списка профилей
[Способ вывода]
Позволяет выбрать метод вывода данных для печати.
[Печать]
Выполняет обычную операцию печати.
[Удержание]
Временно сохраняет данные для печати в очереди удержания принтера. Эта функция удобна, если есть необходимость выполнить пробную распечатку. Наименование данных можно установить в Диалоговое окно [Детали удержания], нажав [Подробно].
Приостановление процесса печати
[Защищенная печать]
Закрепляет за данными для печати a PIN-код (пароль) и сохраняет его в принтере. Эта функция удобна при печати конфиденциальных документов и др. Имя пользователя и PIN-код или пароль можно установить, нажав [Подробно].
Защищенная печать
[Сохранение]
Позволяет сохранять данные для печати в окне принтера. Место сохранения можно установить, нажав [Подробно] (Диалоговое окно [Сохранение деталей]).
Сохранение данных для печати
[Активная печать]
Выполняет задание на печать вне очереди, после того, как будет закончено выполнение текущего задания на печать.
[Правка и предварительный просмотр]
Объединяет несколько файлов в один для распечатки. Можно также просмотреть данные перед печатью и изменить настройки печати. Если выбрано [Блокировать], [Способ вывода] блокируется в режиме [Правка и предварительный просмотр]. Эта функция полезна, если нужно ограничить методы вывода для клиентов в среде общего принтера.
Объединение нескольких файлов
[Блокировать] можно установить только в том случае, если экран драйвера принтера открыт из меню Пуск (см. Изменение настроек по умолчанию).
В начало
[Формат страницы]
Позволяет выбрать формат бумаги, созданный в приложении.
[Размер вывода]
Позволяет выбрать фактический формат бумаги для печати. Если выбран формат, который отличается от настройки [Формат страницы], документ автоматически увеличивается или уменьшается в соответствии с настройкой [Размер вывода].
Масштаб
Если в приложении выбраны несколько форматов страниц, для печати будет использован формат, выбранный в списке [Размер вывода]
Если в списке [Формат страницы] выбран формат, размеры которого превышают поддерживаемые принтером, а в списке [Настройка формата страницы] выбрана настройка [Размер вывода], форматы А0, А1 и А2 будут уменьшены до А3, а форматы В1, В2 и В3 - до В4.
[Количество копий]
Позволяет задать количество копий для распечатки.
Если есть возможность задать количество копий в приложении, задайте количество копий в драйвере принтера или в приложении.
[Ориентация]
Позволяет выбрать ориентацию печати по отношению к направлению подачи страницы через принтер.
В начало
[Разметка страницы]
Позволяет указать количество страниц для распечатки на каждом листе бумаги.
[1 на 1]
Для распечатки одной страницы документа на одном листе бумаги.
[N на 1] (N = 2, 4, 6, 8, 9 или 16)
Для распечатки нескольких страниц (N страниц) на каждом листе бумаги. Выберите порядок расположения печатаемых страниц в меню [Порядок распол. страниц].
[Порядок распол. страниц]
Выберите порядок расположения печатаемых страниц.
Уменьшенный макет
Если в приложении для распечатки был выбран метод сортировки страниц.
При использовании настроек [Ориентация] - [Альбомная] и [Разметка страницы] - [2 на 1] для параметра [Порядок распол. страниц] можно выбрать значение [Сверху вниз/Слева направо]/[Сверху вниз/Справа налево]. Для документов, которые не содержат сочетания страниц в книжной и альбомной ориентации, результат печати не зависит от выбора значения [Сверху вниз/Слева направо] или [Сверху вниз/Справа налево].
При печати документа, который сочетает страницы в книжной и альбомной ориентации, с настройкой [Разметка страницы] - [2 на 1] рекомендуется выбрать настройку [Способ вывода] - [Правка и предварительный просмотр] для отображения диалогового окна [Canon PageComposer] и проверки результата печати в поле [Предварительный просмотр] (см. Объединение нескольких файлов).
[Плакат [1 x 2]]/[Плакат [N x N]] (N = 2, 3 или 4)
Позволяет увеличить данные для печати, представленные на одной странице, и распечатать их на нескольких листах бумаги (по вертикали x по горизонтали). Соединив листы бумаги с распечатанными данными, можно сформировать большой плакат.
[Печать плаката] невозможно выбрать, если установлена настройка [Двухсторонняя печать]. Более подробно о переключении между двухсторонней и односторонней печатью см. в разделе Односторонняя и двухсторонняя печать.
Печать плакатов
[Сведения о плакате]
Позволяет выполнить подробную настройку, если выбрано [Разметка страницы] - [Плакат [1 x 2]] или [Плакат [N x N]] (N = 2, 3 или 4), отображая Диалоговое окно [Сведения о плакате].
[Ручная настройка масштаба]
Позволяет указать коэффициент увеличения/уменьшения для печати. Коэффициент вводится в поле [Масштаб].
Если в приложении предусмотрена возможность настройки масштаба, установите необходимый масштаб в приложении, не устанавливая в драйвере принтера.
[Исп. печать 1-1 для одностор. страниц]
Если при задании этого пункта для параметра [Разметка страницы] выбрано значение [N на 1] (N≥2), одностраничный документ будет печататься один к одному.
[Типу бумаги]
Позволяет выбрать тип бумаги для печати.
[Фоновое изображение]
На распечатываемые страницы можно наложить полупрозрачное фоновое изображение, например [КОНФИДЕНЦИАЛЬНО] или [КОПИЯ].
Выберите [Фоновое изображение] и выберите фоновое изображение для печати в разделе [Имя фонового изображения]. Нажатие [Правка фонового изображения] (Диалоговое окно [Правка фонового изображения]) позволяет создавать и добавлять собственные фоновые изображения.
Печать фонового изображения
В начало
[Специальный формат бумаги]
Диалоговое окно [Настройки специального формата бумаги] позволяет регистрировать и задавать для печати нестандартные форматы бумаги.
Использование специальной бумаги
Данная функция доступна только в том случае, если экран драйвер открыт из меню Пуск (см. Изменение настроек по умолчанию).
[Параметры страницы]
Диалоговое окно [Параметры страницы] позволяет задавать параметры макета (поля, дату, номера страниц и др.) и настройки печати с наложением.
Печать с наложением
[Восс. параметры по умолчанию]
Восстанавливает стандартные (по умолчанию) значения всех настроек.
[Настройки вида]
Позволяет проверить текущие настройки в списке.
[Отображение окна состояния принтера]
Для отображения окна состояния принтера нажмите кнопку []. Окно состояния принтера позволяет проверять состояние принтера, выполнять служебные функции, задавать параметры модуля принтера и т.п.
Подробнее об окне состояния принтера см. в справке, отображаемой с помощью строки меню [Справка].
