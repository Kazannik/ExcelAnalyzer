Диалоговое окно [Настройки функций устройства]
Позволяет установить версию функции принтера, а также необходимость загрузки листов из блока вставки при вставке листов.
Описание в Вкладка [Настройки устройства] (Настройки)
Описание настроек
[Версия функции]
Выберите версию функции, имеющуюся на странице конфигурации, которую можно распечатать с панели управления принтера.
[Выполняется загрузка из устройства вставки [Листы для вставки]]
Позволяет использовать блок вставки для функции [Листы для вставки].
