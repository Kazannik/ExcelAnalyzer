Диалоговое окно [Детали защищенной печати]
Позволяет настроить информацию пользователя для защищенной печати.
Описание в Функции, предлагаемые на всех вкладках (настройки)
Процедура для Защищенная печать
Описание настроек
[Имя пользователя]
Позволяет ввести имя пользователя для данных печати.
[PIN-код]/[Пароль]
Позволяет ввести PIN-код или пароль длиной до 7 символов.
