Описание в Вкладка [Макет страницы] (Настройки)
Процедура для Печать с наложением
Описание настроек
Вкладка [Параметры макета]
Настройки отображаются в окне предварительного просмотра.
[Контроль края]
Добавляет рамку на каждую распечатываемую страницу. При добавлении рамки данные печати уменьшаются настолько, чтобы уместиться в рамке на странице.
[Печать даты], [Имя пользователя печати]/[Печатать имя для входа], [Печать номеров страниц]
Позволяет добавить дату, имя для подключения к компьютеру и номер каждой распечатываемой страницы. Можно установить расположение каждого их указанных элементов.
[Настройки формата]
Позволяет задать настройки шрифта для печати даты, имени пользователя, имени для входа и/или номеров страниц.
[Восс. параметры по умолчанию]
Восстанавливает стандартные (по умолчанию) значения всех настроек.
Вкладка [Наложение]
Процедура для Печать с наложением
[Способ обработки]
[Не использовать печать с наложением]
Выполняется обычная печать без использования функции наложения.
[Использовать печать с наложением]
Подготовленный предварительно файл формы и данные для печати печатаются вместе, с наложением друг на друга. Выберите файл формы в списке [Список файлов]. Если информацию из принтера получить невозможно, [Список файлов] не отображается. В таком случае введите имя файла в поле [Имя файла].
При черно-белой печати невозможно использовать форму, созданную в цветном режиме.
[Создать файл формы]
Данные для печати сохраняются в принтере как файл формы. Введите имя файла формы в поле [Имя файла].
Указанные настройки файла формы и данных для печати должны совпадать:
[Макет страницы] → [Размер вывода]
[Макет страницы] → [Ориентация]
[Применить к]
При выполнении двухсторонней печати позволяет указать диапазон печати с наложением.
[Восс. параметры по умолчанию]
Восстанавливает стандартные (по умолчанию) значения всех настроек.
