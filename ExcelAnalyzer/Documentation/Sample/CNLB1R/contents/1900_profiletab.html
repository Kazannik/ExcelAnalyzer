Как открыть вкладку [Профиль]:
[+][-]
Нажмите [] (Пуск) → [Устройства и принтеры].
Выше указаны действия в ОС Windows 7 и Server 2008 R2.
В ОС Windows 10: нажмите [Параметры] → [Устройства] → [Принтеры и сканеры] → [Сопутсвующие параметры] → [Устройства и принтеры].
В ОС Windows 8, 8.1, Server 2012 и Server 2012 R2: на рабочем столе выдвиньте боковую панель с правой стороны экрана и выберите пункты [Параметры] → [Панель управления] → [Устройства и принтеры].
В ОС Windows Vista и Server 2008: нажмите [] (Пуск) → [Панель управления] → [Принтеры].
В ОС Windows Server 2003 и Server 2003 R2: нажмите [Пуск] → [Принтеры и факсы] (или [Пуск] → [Панель управления] → [Принтеры и другое оборудование] → [Принтеры и факсы]).
Щелкните правой кнопкой мыши значок используемого притера → нажмите [Свойства принтера] (или [Свойства]).
Выберите вкладку [Профиль].
Описание настроек
[Список профилей]
Отображает список предварительно зарегистрированных профилей и профилей, добавленных во вкладку [Профиль].
Профили, отмеченные красной точкой (), являются предварительно зарегистрированными профилями и не подлежат правке или удалению.
Профили, добавленные из вкладок [Макет страницы], [Окончательная обработка], [Источник бумаги] и [Качество], не отображаются.
[Настройки вида]
Нажмите на вкладку, чтобы отобразить список настроек.
[Имя], [Комментарий]
Отображают название и комментарий профиля, выбранного в списке [Список профилей].
[Добавление]
Позволяет добавить новый профиль в Диалоговое окно [Добавление/изменение профиля].
[Правка]
Позволяет изменять настройки зарегистрированного пользователем профиля в Диалоговое окно [Добавление/изменение профиля].
[Удалить]
Удаляет профиль, выбранный в списке [Список профилей].
[Импорт]
Импортирует профиль, сохраненный в виде файла «*.cfg», и добавляет его в список [Список профилей].
[Экспорт]
Экспортирует профиль, выбранный в списке [Список профилей], и сохраняет его в виде файла «*.cfg».
[Определение свойств документа]
Разрешает определение настроек на экране настроек печати драйвера принтера. Данная функция полезна, если нужно ограничить использование настроек печати на клиентских компьютерах в среде общего принтера. При отказе от данных настроек на сервере настройки всех клиентских компьютерах фиксируются на заданных на сервере.
[Разрешить выбор профиля]
Позволяет выбирать профили во вкладках [Основные настройки], [Макет страницы], [Окончательная обработка], [Источник бумаги] и [Качество].
[Разрешить изменение настроек]
Данную настройку можно выбрать, если выбрано [Разрешить выбор профиля]. Позволяет задавать настройки печати, кроме выбора профилей во вкладках [Основные настройки], [Макет страницы], [Окончательная обработка], [Источник бумаги] и [Качество].
[О программе]
Отображает версию драйвера принтера.
