Управление именем подразделения
Вы можете осуществлять управление устройством, используя имя и PIN-код подразделения.
О данной функции
1.
Откройте вкладку [Настройки устройства].
[+][-]
Нажмите [] (Пуск) → [Устройства и принтеры].
Выше указаны действия в ОС Windows 7 и Server 2008 R2.
В ОС Windows 10: нажмите [Параметры] → [Устройства] → [Принтеры и сканеры] → [Сопутсвующие параметры] → [Устройства и принтеры].
В ОС Windows 8, 8.1, Server 2012 и Server 2012 R2: на рабочем столе выдвиньте боковую панель с правой стороны экрана и выберите пункты [Параметры] → [Панель управления] → [Устройства и принтеры].
В ОС Windows Vista и Server 2008: Нажмите [] (Пуск) → [Панель управления] → [Принтеры].
В ОС Windows Server 2003 и Server 2003 R2: нажмите [Пуск] → [Принтеры и факсы] (или [Пуск] → [Панель управления] → [Принтеры и другое оборудование] → [Принтеры и факсы]).
Щелкните правой кнопкой мыши значок принтера, который Вы желаете определить → нажмите [Свойства принтера] (или [Свойства]).
Выберите вкладку [Настройки устройства].
2.
Выберите [Использовать управление именем подразделения] или выберите [Управление пользователями] - [Управление именем подразделения] → нажмите [Настройки].
Откроется Диалоговое окно [Настройки имени подразделения/PIN-кода]/[Настройки имени подразделения/пароля].
3.
Выберите [Разрешить настройки PIN-кода]/[Разрешить настройки пароля] → введите [Имя подразделения] и [PIN-код]/[Пароль].
Для подтверждения настройки принтера: Нажмите [Проверка].
Для подтверждения имени и PIN-кода подразделения каждый раз при печати: Выберите [Подтв. имя подразделения/PIN-код во время печати].
4.
Нажмите [ОК] → нажмите [Применить] во вкладке [Настройки устройства].
5.
Нажмите кнопку печати в приложении.
6.
В случае отображения диалогового окна [Подтвердить имя подразделения/PIN-код]/[Подтвердить имя подразделения/пароль], подтвердите настройки.
Начнется процесс печати.
В начало
О данной функции
Поддерживаемые принтеры
Принтер, подключенный к сети
Необходимые условия
Учетная запись с разрешением на полный доступ к принтеру
Установка Canon Driver Information Assist Service (можно установить с помощью программы установки драйвера принтера)
Справки
Если в диалоговом окне [Настройки имени подразделения/PIN-кода]/[Настройки имени подразделения/пароля] не указан PIN-код/пароль, каждый раз при печати Вам придется вводить PIN-код/пароль.
Если имя подразделения, PIN-код или пароль, которые введены в диалоговом окне [Настройки имени подразделения/PIN-кода]/[Настройки имени подразделения/пароля] или диалоговом окне [Подтвердить имя подразделения/PIN-код]/[Подтвердить имя подразделения/пароль] не распознаются, см. раздел Не удается определить состояние принтера либо использовать функцию управления именем подразделения или проверку подлинности пользователя.
