Диалоговое окно [Сохранение деталей]
Позволяет установить наименование данных и место для сохранении данных для печати. Данные для печати сохраняются в папке в принтере, и печатаются по мере надобности.
Описание в Функции, предлагаемые на всех вкладках (настройки)
Процедура для Сохранение данных для печати
Описание настроек
[Наименование данных]
Позволяет установить наименование данных для печати.
[Почтовые ящики]/[Папки входящей почты пользователя]/[Почтовый ящик]
Позволяет указать номер почтового ящика/папки входящей почты пользователя/папки назначения.
Чтобы выбрать несколько папок для сохранения:
Отдельные номера: удерживая нажатой кнопку [Ctrl], выберите соответствующие номера
Последовательные номера: удерживая нажатой кнопку [Shift], выберите первый и последний номера
