Описание в Вкладка [Источник бумаги] (Настройки)
Процедура для Вставка листов
Описание настроек
[Список настроек]
Отображает в списке такие настройки как тип бумаги для вставки и перед какой страницей следует вставить эту бумагу.
[Листы для вставки]
Выберите тип листа для вставки в документ из следующих вариантов:
[Вставленные листы]
Производит вставку листов для разделения документа на разделы.
[Страницы главы]
Вставка страниц главы для разделения документа на главы.
[Бумага для таблиц]
Вставка бумаги для таблиц в документ. Можно также указать тип бумаги для таблиц.
В начало
[Источник бумаги]
Позволяет указать источник бумаги для вставленных листов, страниц главы и бумаги для таблиц.
[Печать на]
Позволяет указать, следует ли печатать на вставленных листах, страницах главы или бумаге для таблиц.
[Нет]
Печать на вставленных листах не выполняется.
[Только передней стороне]
Печать производится только на передней стороне вставленных листов бумаги.
[Только оборотной стороне]
Печать производится только на задней стороне вставленных листов бумаги.
[По обеим сторонам]
Печать на обеих сторонах вставленных листов бумаги.
Если для параметра [Листы для вставки] выбрано значение [Страницы главы], выбирается пункт [Только передней стороне] и печатать можно только на передней стороне вставленных листов бумаги.
Если для параметра [Листы для вставки] выбрано значение [Бумага для таблиц], можно выбрать [Нет] или [Только передней стороне].
[Вставка]
Позволяет указать номера страниц, перед которыми добавлять вставленные листы, страницы главы или бумагу для таблиц.
[Бумага для таблиц]
Нажмите [Настройки], чтобы выбрать тип бумаги для таблиц, и нажмите [Сведения о бумаге для таблиц], чтобы настроить, насколько далеко следует сместить бумагу при печати на бумаге для таблиц.
[Удалить]
Удаляет выбранные настройки вставки листа из списка [Список настроек].
[Добавить]
Добавляет настройки вставки листа в [Список настроек].
