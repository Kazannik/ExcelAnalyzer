Как открыть вкладку [Настройки устройства]:
[+][-]
Нажмите [] (Пуск) → [Устройства и принтеры].
Выше указаны действия в ОС Windows 7 и Server 2008 R2.
В ОС Windows 10: нажмите [Параметры] → [Устройства] → [Принтеры и сканеры] → [Сопутсвующие параметры] → [Устройства и принтеры].
В ОС Windows 8, 8.1, Server 2012 и Server 2012 R2: на рабочем столе выдвиньте боковую панель с правой стороны экрана и выберите пункты [Параметры] → [Панель управления] → [Устройства и принтеры].
В ОС Windows Vista и Server 2008: Нажмите [] (Пуск) → [Панель управления] → [Принтеры].
В ОС Windows Server 2003 и Server 2003 R2: нажмите [Пуск] → [Принтеры и факсы] (или [Пуск] → [Панель управления] → [Принтеры и другое оборудование] → [Принтеры и факсы]).
Щелкните правой кнопкой мыши значок принтера, который Вы используете → нажмите [Свойства принтера] (или [Свойства]).
Выберите вкладку [Настройки устройства].
Описание настроек
Настройки параментов принтера
([Параметры источника бумаги], [Параметры вывода], [Количество секций], [Устройство вставки документа]/[Устройство вставки], [Блок двухсторонней печати], [Установленный жесткий диск], [3-сторонний блок], [Установленная SD карточка], [Функция защищенного фонового изображения устройства]/[Функция защищенной печати устройства])
Вы можете задать параметры, установленные на принтере. Настройка данных параметров позволяет печатать, используя соответствующие функции. Нажатие [Опред-е сост. устройства] отображает информацию об устройстве и автоматически задает параметры. Если информацию об устройстве получить невозможно, настройте параметры вручную.
Настройка параметров печати
[Буферизация на хост-компьютере]
Позволяет выбрать, будет ли обработка данных печати выполняться на хост-компьютере или на принтере с отключенной обработкой на хост-компьютере. При нажатии [Авто], обработка осуществляется на принтере и возможна при использовании только функций принтера, увеличивая скорость печати.
При выборе [Отключена], Вы не сможете настраивать следующие функции.
Вкладка [Основные настройки] или [Макет страницы] → [Разметка страницы] → [N на 1] (кроме 1, 2 и 4 на 1)
Вкладка [Макет страницы] → [Фоновое изображение]
Вкладка [Макет страницы] → [Параметры страницы] → [Параметры макета]
Вкладка [Основные настройки] или [Окончательная обработка] → [Буклет] → [Процесс печати буклета] → [С настройками драйвера]
Вкладка [Окончательная обработка] → [Дополнительные настройки] → [Лицевой стороной вверх/вниз] → [Лицевой вверх [Обр. порядок вывода]]
Вкладка [Источник бумаги] → [Выбор бумаги]
Вкладка [Качество] → [Дополнительные настройки] → [CanoFine] → [Вкл.]
Итоговый результат (внешний вид) печати буклета отличается при выборе [Авто] или [Активно], а также при выборе [Отключена].
[Функция защищенного фонового изображения устройства]/[Функция защищенной печати устройства]
Выберите данный параметр, если Вы используете комплект расширения блокировки заданий. Он включает в данные для печати информации об ограничении и отслеживании в виде точечного изображения, что позволяет пресечь утечку конфиденциальных документов. Нажатие [Опред-е сост. устройства] отображает информацию об устройстве и автоматически задает данный параметр.
Необходимые условия:
Выберите вкладку [Окончательная обработка] → [Дополнительные настройки] → [Функция защищенного фонового изображения устройства]/[Функция защищенной печати устройства]
Включение функции защищенной печати принтера
Функции, недоступные в сочетании с данной функцией:
Вкладка [Макет страницы] → [Параметры страницы] → вкладка [Наложение] → [Способ обработки] →[Использовать печать с наложением]
Вкладка [Основные настройки] или [Окончательная обработка] → [Буклет] → [Процесс печати буклета] → [С настройками устройства]
Вкладка [Окончательная обработка] → [Дополнительные настройки] → [Нумерация копий]
Вкладка [Окончательная обработка] → [Стиль печати] → [Бесшвейное скрепление [1-сторонняя]]/[Бесшвейное скрепление [2-сторонняя]]
Вкладка [Качество] → [Дополнительные настройки] → [Экономия тонера]
[Использовать управление именем подразделения]
Позволяет управлять печатью с именами и паролями подразделений. Чтобы задать расширенные настройки, нажмите [Настройки] (Диалоговое окно [Настройки имени подразделения/PIN-кода]/[Настройки имени подразделения/пароля]).
Управление именем подразделения
[Задать информацию пользователя]
Позволяет задать имя пользователя для данных печати и PIN-код (пароль) для защищенной печати. Чтобы задать расширенные настройки, нажмите [Настройки] (Диалоговое окно [Настройки информации пользователя]).
Настройки информации пользователя
[Разрешить использовать защищенную печать]
Активирует [Защищенная печать] в качестве параметра для [Способ вывода].
[Указать номер почтового ящика для сохранения]/[Указать номер входящей почты для сохранения]/[Указать номер почтового ящика для сохранения]
Выберите данный параметр, если при сохранении данных печати Вы желаете указать имя и место сохранения данных.
[Управление пользователями]
Если выбран режим [Управление именем подразделения]: задайте подробные настройки управления именем подразделения в пункте [Настройки] (диалоговое окно [Настройки имени подразделения/PIN-кода]/[Настройки имени подразделения/пароля]).
Если выбран режим [Проверка подлинности пользователя]: Задайте подробные настройки проверки подлинности пользователя в пункте [Настройки] (диалоговое окно [Настройки проверки подлинности пользователя]).
[Подробно]
В Диалоговое окно [Расширенные настройки] (Вкладка [Настройки устройства]) можно задать дополнительные настройки драйвера, такие как специальные сведения о бумаге, названия заданий для данных печати и т.д.
[Функция]
В Диалоговое окно [Настройки функций устройства] можно настроить [Идентификатор функции контроллера], [Версия функции] и [Выполняется загрузка из устройства вставки [Листы для вставки]].
[Формат для назначения лотка]
Диалоговое окно [Формат для назначения лотка] позволяет подтвердить и задать формат и тип бумаги, установленные в каждом источнике бумаги.
Настройки источника бумаги
[Инф-ия о расширениях]
Отображает информацию о модулях расширения (расширенных функциях), добавленных в драйвер принтера.
[Опред-е сост. устройства]
Получение информации из принтера и применение ее к перечисленным ниже настройкам.
