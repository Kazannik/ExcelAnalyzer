Основная процедура печати
Ниже описан основной метод печати с использованием драйвера принтера.
1.
В меню приложения выберите [Печать].
Откроется диалоговое окно [Печать].
При печати из приложения Windows Store или Windows можно настроить только некоторые из функций печати.
В ОС Windows 8/8.1/Server 2012/Server 2012 R2 при печати из приложения Windows Store или Windows выдвиньте боковую панель с правой стороны экрана и нажмите [Устройства] (→ [ Печать]) → используемый принтер → [Печать].
2.
Выберите имя используемого принтера → нажмите [Свойства принтера] (или [Свойства]).
Отобразится экран параметров печати драйвера принтера.
3.
Задайте [Количество копий] во вкладке [Основные настройки].
4.
Выберите формат страницы, на котором нужно печатать, в списке [Размер вывода].
Если выбран формат, который отличается от выбранного в [Формат страницы] (формат страницы для данных печати), данные печати будут масштабированы в соответствии с форматом страницы [Размер вывода] (см. Масштаб).
5.
Задайте другие необходимые настройки.
Для печати данных с нескольких страниц на одном листе: выберите [Разметка страницы] → [N на 1]
Для переключения между односторонней и двухсторонней печатью: выберите [Односторонняя печать] или [Двухсторонняя печать]
Переходя с вкладки на вкладку, настройте необходимые функции.
Функции, предлагаемые на всех вкладках (что можно сделать)
Вкладка [Макет страницы] (что можно сделать)
Вкладка [Окончательная обработка] (что можно сделать)
Вкладка [Источник бумаги] (что можно сделать)
Вкладка [Качество] (что можно сделать)
6.
Нажмите [ОК] на экране параметров печати драйвера принтера.
7.
Нажмите [Печать] (или [OK]).
Начнется процесс печати.
Справки
Если не удается установить формат страницы, ориентацию печати и другие параметры в драйвере принтера, установите их, выбрав [Настройка печати], [Макет страницы] или другую соответствующую команду в меню [Файл] приложения.
В ОС Windows 8/8.1/Server 2012/Server 2012 R2 при печати из приложения Windows Store, если ранее были выбраны настройки, которые требуют ввода или выполнения какой-либо процедуры при печати в режиме защищенной печати, правки + просмотра, управления именем подразделения и т.п., отображается сообщение [Принтер требует вашего внимания. Перейдите к рабочему столу.]. В таком случае перейдите на рабочий стол и следуйте указаниям в отображаемом диалоговом окне.
