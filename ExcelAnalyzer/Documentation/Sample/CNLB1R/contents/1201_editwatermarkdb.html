Диалоговое окно [Правка фонового изображения]
Позволяет создать фоновые изображения с желаемым текстом. Можно отрегулировать размер, цвет и место расположения фонового изображения.
Описание в Вкладка [Макет страницы] (Настройки)
Процедура для Печать фонового изображения
Описание настроек
[Список фоновых изображений]
Отображает зарегистрированные фоновые изображения в виде списка. При выборе в списке фонового изображения оно предварительно отображается слева.
Фоновые изображения, отмеченные красной точкой (), являются предварительно зарегистрированными фоновыми изображениями и не подлежат правке или удалению.
Зарегистрировать можно до 50 пользовательских фоновых изображений.
[Добавить]
При нажатии на данную кнопку в [Список фоновых изображений] добавляется новое фоновое изображение [Без имени]. Вы можете задать название фонового изображения в [Имя] и его настройки во вкладках [Атрибуты], [Выравнивание] и [Стиль печати].
[Удалить]
При выборе фонового изображения и нажатии этой кнопки фоновое изображение удаляется.
[Имя]
Отображает название фонового изображения, выбранного в списке [Список фоновых изображений]. При добавлении нового фонового изображения введите его название здесь. Названия зарегистрированных пользователями фоновых изображений можно корректировать.
Вкладка [Атрибуты]
Позволяет задавать настройки, связанные с текстом фонового изображения. В поле [Текст] введите строку символов, которая будет печататься в фоновом изображении.
Если используется черно-белый принтер, выбранный для параметра [Цвет] цвет на печатном листе передается оттенками серого.
Вкладка [Выравнивание]
[Положение] позволяет координатами указать место расположения фонового изображения по отношению к центру (0) документа. [Наклон] позволяет установить угол наклона фонового изображения.
Вкладка [Стиль печати]
[Рамка фонового изображения] позволяет установить, будет ли печататься вокруг фонового изображения рамка. [Положение фонового изображения] позволяет установить место наложения по отношению к печатным данным. При выборе [Печать только на первой странице] фоновое изображение печатается только на первой странице.
