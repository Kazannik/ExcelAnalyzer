Установка метода окончательной обработки
При распечатке нескольких копий можно установить порядок расположения страниц (разбор/группирование) и следует ли их сшивать.
Разбор
Группирование
Офсет
Поворот
Пробивание отверстий
Сшивание
О данной функции
1.
Откройте вкладку [Основные настройки] или [Окончательная обработка].
2.
Задайте необходимые настройки в [Окончательная обработка].
Для вывода страниц в порядке их расположения: выберите [Разобрать] (выберите [Сшивание + pазобрать]/[Эко (без сшивания) + разобрать]/[Сшивание и pазобрать], если желаете сшить)
Для вывода страниц группами, состоящими из страниц с одним и тем же номером: выберите [Группирование] (выберите [Сшивание + группирование]/[Эко (без сшивания) + группирование]/[Сшивание и группирование], если желаете сшить)
Для установки сдвига разобранных по копиям/сгруппированных страниц: выберите [Офсет] ([Количество копий для офсета] также может быть указан метод сортировки страниц)
Для вывода каждой копии документа под углом 90 градусов к предыдущей и следующей копиям: выберите [Поворот]
3.
Если в бумаге требуется пробить отверстия, выберите вкладку [Окончательная обработка] → [Окончательная обработка] → [Пробивание отверстий] → [Вкл.].
В начало
О данной функции
Необходимые параметры
Сдвиг: финишер
Сшивание: параметр вывода для сшивания
Эко (Без сшивания): опции вывода для переплета без сшивания
Пробивание отверстий: финишер и перфораторное устройство
Функция поворота доступна только в том случае, если не подключен финишер.
Форматы бумаги, подлежащие перфорированию и выводу
Если для расположения переплета выбран короткий край: A3, B4, A4, B5, Letter, Legal, 11x17
Если для расположения переплета выбран длинный край: A4, B5, Letter
Справки
Возможные места для сшивания зависят от формата бумаги, направления подачи бумаги и источника бумаги.
Одна копия или первая страница документа при выводе сдвигается согласно выбранному параметру вывода.
Возможное число пробиваемых дырок варьируется в зависимости от типа перфораторного устройства, размера бумаги и ориентации напечатанных страниц.
[Пробивание отверстий] можно задавать только во вкладке [Окончательная обработка].
